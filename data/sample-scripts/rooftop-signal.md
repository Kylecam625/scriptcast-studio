# The Rooftop Signal

[Rain lashes the old radio tower. A warning light blinks red.]

MARA: We only get one chance to send the signal.

JULES: Then make it loud.

NARRATOR: Jules pulls a cracked transmitter from the supply case.

MARA: If anyone is listening, they need to know where we are.

JULES: And if the wrong people hear it?

MARA: Then we run faster than they do.
